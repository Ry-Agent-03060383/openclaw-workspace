# 智慧金服系统 - 中小企业融资服务平台

## 技术栈

- **后端**: Spring Boot 3.2 + Spring Security + JWT
- **前端**: Vue 3 + Vite + Element Plus
- **数据库**: MySQL 8.0 + Redis
- **AI**: OpenAI API / 国产大模型集成

## 模块列表

1. 用户系统（企业、银行、第三方、运营）
2. 银企对接
3. 征信评级
4. 助贷服务
5. 贷后监控
6. 风险处置
7. AI客服

## 快速启动

### 后端
```bash
cd backend
./mvnw spring-boot:run
```

### 前端
```bash
cd frontend
npm install
npm run dev
```
